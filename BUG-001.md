# BUG-001 – Mensaje incorrecto al ingresar RUT válido en login

## Severidad:
Media

## Prioridad:
Media

## Descripción:
Al intentar iniciar sesión con un RUT válido y correctamente formateado, el sistema muestra un mensaje de error genérico, a pesar de que los datos ingresados cumplen con el formato esperado.
Esto genera confusión en el usuario, ya que no se informa claramente la causa del error ni cómo resolverlo.

## Pasos para reproducir:
	1.	Ingresar a la página de login en https://bancoficticio.demo/login.
	2.	Ubicar el campo “RUT o usuario”.
	3.	Ingresar un RUT válido con formato correcto (ejemplo: 12.345.678-9).
	4.	Ingresar una contraseña válida en el campo “Contraseña”.
	5.	Hacer clic en el botón “Ingresar”.
	6.	Observar el mensaje mostrado por el sistema.


## Resultado esperado:
	•	El sistema debe permitir el inicio de sesión al ingresar un RUT válido y una contraseña correcta.
	•	No se debe mostrar ningún mensaje de error.


## Resultado actual:
	•	El sistema muestra un mensaje de error genérico e impreciso, indicando que existe un problema sin especificar la causa, a pesar de que el RUT ingresado es válido y la contraseña es correcta.
