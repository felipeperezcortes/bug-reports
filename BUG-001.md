# BUG-001 - Mensaje incorrecto al ingresar RUT válido en login

## Severidad: 
Media

## Prioridad: 
Media

## Descripción
Cuando el usuario ingresa un RUT válido en la página de login, 
el sistema muestra un mensaje genérico indicando error, a pesar de que el formato es correcto. 
Esto genera confusión en el usuario, ya que no existe información clara sobre la causa del error.

## Pasos para reproducir
1. Ingresar a la página de login en https://bancoficticio.demo/login.
2. Localizar el campo "RUT o usuario".
3. Ingresar un RUT válido con su formato correcto. (Ejemplo: 12.345.678-9)
4. Ingresar una contraseña válida en el campo "Contraseña".
5. Hacer clic en el botón "Ingresar".
6. Observar el mensaje de error genérico que muestra el sistema.

## Resultado esperado
El usuario debería poder iniciar sesión al ingresar un RUT válido en el formato correcto 
junto con su contraseña correcta. El sistema no debería mostrar ningún mensaje de error.

## Resultado actual
A pesar de ingresar un RUT válido y una contraseña correcta, el sistema muestra 
un mensaje de error genérico y poco claro, indicando que existe un problema sin especificar.
