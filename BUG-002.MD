## BUG-002 - Mensaje incorrecto al intentar hacer login sin ingresar el RUT.

## Severidad
Media

## Prioridad
Media

## Descripción
El usuario intenta iniciar sesión sin ingresar su RUT, pero sí ingresa una contraseña válida. 
El sistema muestra un mensaje genérico de “Error desconocido”, en lugar de indicar que falta ingresar el RUT 
y mostrar el formato correcto. Esto confunde al usuario e impide entender cuál es el problema real.

## Pasos para reproducir
1. Ingresar a la página de login en https://bancoficticio.demo/login.
2. Localizar el campo "RUT o usuario".
3. Dejar el campo "RUT o usuario" vacío.
4. Ingresar una contraseña válida en el campo "Contraseña".
5. Hacer clic en el botón "Ingresar".
6. Observar el mensaje de error mostrado por el sistema.

## Resultado esperado
El sistema debería mostrar un mensaje claro indicando que falta ingresar el RUT
(por ejemplo: "Debes ingresar tu RUT") y, opcionalmente, mostrar un ejemplo del formato correcto.

## Resultado actual
El sistema muestra un mensaje genérico de "Error desconocido".
