# BUG-002 – Mensaje incorrecto al intentar iniciar sesión sin ingresar el RUT

## Severidad:
Media

## Prioridad:
Media

## Descripción:
Al intentar iniciar sesión sin ingresar el RUT, pero proporcionando una contraseña válida, el sistema muestra un mensaje genérico de “Error desconocido”, en lugar de indicar que el campo RUT es obligatorio.
Esto genera confusión en el usuario, ya que no se informa claramente cuál es el dato faltante ni cómo corregir el error.

## Pasos para reproducir:
	1.	Ingresar a la página de login en https://bancoficticio.demo/login.
	2.	Ubicar el campo “RUT o usuario”.
	3.	Dejar el campo “RUT o usuario” vacío.
	4.	Ingresar una contraseña válida en el campo “Contraseña”.
	5.	Hacer clic en el botón “Ingresar”.
	6.	Observar el mensaje mostrado por el sistema.

## Resultado esperado:
	•	El sistema debe impedir el inicio de sesión.
	•	Se debe mostrar un mensaje claro indicando que el campo RUT es obligatorio (por ejemplo: “Debes ingresar tu RUT”).
	•	Opcionalmente, el mensaje puede incluir un ejemplo del formato correcto esperado.

## Resultado actual:
	•	El sistema muestra un mensaje genérico de “Error desconocido”, sin indicar que el problema corresponde a la ausencia del RUT.
