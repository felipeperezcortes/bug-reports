# BUG-003 - Mensaje incorrecto al intentar realizar una transferencia sin el saldo suficiente

## Descripción
Cuando el usuario intenta realizar una transferencia sin contar con el saldo suficiente, 
el sistema muestra un mensaje de error genérico y poco claro, lo que genera confusión.

# Severidad
Media

# Prioridad
Media

## Pasos para reproducir
1. Ingresar al portal del banco ficticio y realizar login correctamente.
2. Acceder a la sección "Transferencias" desde el menú principal.
3. Seleccionar la cuenta de origen.
4. Ingresar un monto que excede el saldo disponible en la cuenta.
5. Presionar el botón "Transferir".
6. Observar el mensaje genérico de error que muestra el sistema.

## Resultado esperado
El sistema debería mostrar un mensaje claro indicando que el usuario no cuenta con 
el saldo suficiente para realizar la transferencia. 
Por ejemplo: "No cuentas con el saldo suficiente para completar esta operación."

## Resultado actual
El sistema muestra un mensaje genérico de "Error de sistema", 
que no explica que el problema es la falta de saldo.
