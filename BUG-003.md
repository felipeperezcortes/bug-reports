# BUG-003 – Mensaje incorrecto al intentar realizar una transferencia con saldo insuficiente

## Severidad:
Media

## Prioridad:
Media

## Descripción:
Al intentar realizar una transferencia ingresando un monto mayor al saldo disponible en la cuenta de origen, el sistema muestra un mensaje genérico de error, en lugar de indicar claramente que el problema corresponde a saldo insuficiente.
Esto genera confusión en el usuario y no entrega información útil para corregir la operación.

## Pasos para reproducir:
	1.	Ingresar al portal del banco ficticio y realizar login correctamente.
	2.	Acceder a la sección “Transferencias” desde el menú principal.
	3.	Seleccionar la cuenta de origen.
	4.	Ingresar un monto que excede el saldo disponible en la cuenta.
	5.	Presionar el botón “Transferir”.
	6.	Observar el mensaje mostrado por el sistema.

## Resultado esperado:
	•	El sistema debe impedir la ejecución de la transferencia.
	•	Se debe mostrar un mensaje claro indicando que el usuario no cuenta con saldo suficiente para realizar la operación
(por ejemplo: “No cuentas con el saldo suficiente para completar esta operación”).

## Resultado actual:
	•	El sistema muestra un mensaje genérico de “Error de sistema”, sin indicar que la causa del problema es la falta de saldo en la cuenta de origen.
