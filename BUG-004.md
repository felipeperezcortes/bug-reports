## BUG-004 - Mensaje de éxito al actualizar correo, pero se mantiene el correo antiguo

## Severidad
Media

## Prioridad
Media

## Descripción
El usuario intenta actualizar su correo electrónico y el sistema muestra un mensaje indicando 
que la actualización fue exitosa. Sin embargo, al revisar los datos personales, 
el correo sigue siendo el mismo correo original, por lo que la actualización no se refleja realmente.

## Pasos para reproducir
1. Ingresar al portal del banco ficticio.
2. Realizar login correctamente ingresando un root válido y una contraseña correcta.
3. Acceder a la sección “Datos personales” o “Actualizar datos”.
4. Hacer clic en la opción “Actualizar correo”.
5. Ingresar un nuevo correo electrónico.
6. Confirmar la actualización.
7. Observar el mensaje del sistema indicando que el correo fue actualizado exitosamente.
8. Volver a la sección de datos personales y verificar el correo almacenado.

## Resultado esperado
El sistema debería mostrar el mensaje de actualización exitosa 
y reflejar el nuevo correo electrónico en la sección de datos personales.

## Resultado actual
El sistema muestra el mensaje de “correo actualizado exitosamente”, 
pero el correo electrónico sigue siendo el mismo correo original y no se refleja la actualización.
