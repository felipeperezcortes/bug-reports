## BUG-004 – Mensaje de éxito al actualizar correo electrónico, pero se mantiene el correo anterior

## Severidad:
Media

## Prioridad:
Media

## Descripción:
Al intentar actualizar el correo electrónico desde la sección de datos personales, el sistema muestra un mensaje indicando que la actualización fue exitosa. Sin embargo, al revisar nuevamente los datos personales, el correo electrónico no se actualiza y se mantiene el valor anterior.
Esto indica un problema de persistencia de datos, ya que el sistema confirma la operación sin reflejar realmente el cambio.

## Pasos para reproducir:
	1.	Ingresar al portal del banco ficticio.
	2.	Realizar login correctamente ingresando un RUT válido y una contraseña correcta.
	3.	Acceder a la sección “Datos personales” o “Actualizar datos”.
	4.	Hacer clic en la opción “Actualizar correo”.
	5.	Ingresar un nuevo correo electrónico válido.
	6.	Confirmar la actualización.
	7.	Observar el mensaje del sistema indicando que el correo fue actualizado exitosamente.
	8.	Volver a la sección de datos personales y verificar el correo almacenado.


## Resultado esperado:
	•	El sistema debe mostrar un mensaje indicando que el correo fue actualizado exitosamente.
	•	El nuevo correo electrónico debe quedar correctamente actualizado y persistido en la sección de datos personales.

## Resultado actual:
	•	El sistema muestra el mensaje de “correo actualizado exitosamente”, pero el correo electrónico permanece sin cambios y continúa mostrando el valor anterior.
