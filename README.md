# Bug Reports – Práctica QA Manual

Este repositorio contiene cuatro bug reports escritos en Markdown como parte de mi proceso de aprendizaje para QA Manual Junior.

Cada bug report describe un problema funcional detectado a partir de casos de prueba simples:
- una descripción breve del error,
- los pasos para reproducirlo,
- la diferencia entre el resultado esperado y el resultado actual,
- su severidad y prioridad.
